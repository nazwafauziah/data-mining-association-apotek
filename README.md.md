## 📌 Deskripsi Project
Project ini bertujuan untuk menemukan pola asosiasi antar produk obat yang sering dibeli
bersamaan dalam satu transaksi di apotek, menggunakan algoritma **Apriori** dengan
pendekatan framework **CRISP-DM**.
